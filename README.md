# Клиентская часть (Фронт) основного сервиса


**Если стенд поднимается локально:**
- Предварительно необходимо собрать инфраструктуру из репозитория extinfrastructure и бэк из репозитория mainservicebackend. В случае необходимости выполнять расчеты сообраться из репозитория computeservices.

**Если стенд (extinfrastructure, mainservicebackend и computeservices) поднимается в облаке:**
- Редактировать хосты в [app_config.json](public/app_config.json)

**Инструкиця по сборке в контейнере:**
- выполнить `docker compose up --build`

**Инструкиця по сборке без контейнера:**
- Предустановленный nodejs 20.x
- Устанавливаем зависимости находясь в корне `npm install`
- Запускаемся находясь в корне `npm start`

**Прежде чем писать код:** 
- Необходимо установить расширение Biome для своей IDE.
- Установить Biome глобально для всей системы `npm i -g @biomejs/biome`
- Ознакомиться с документацией Feature-Sliced Design.

**По зависимостям**

В идеале не устанавливать ничего нового. Чем меньше подзависимостей в проекте тем меньше потенциальной поддерки на ПСИ. Вот работы которые уже пришлось проводить в связи с алертами сканера:

- eslint всей экосистемой заменен на biome в связи с protestware в подзависимости acorn с версии 8.7.1
